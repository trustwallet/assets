## Overview

BEK (Oybek and Otabek Token) is a next-generation ERC-20 utility token built on the Polygon blockchain. With a fixed supply of 56,000,000 BEK tokens, it is designed for integration into national digital economy infrastructure.

BEK aims to facilitate transparent, secure, and efficient government-level financial operations by 2030.
